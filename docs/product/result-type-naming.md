# 长发男鉴定器结果类型命名决策

状态：暂定正式短类型名

更新时间：2026-06-30

## 决策背景

新版移动端方案将结果名拆成两层：

- **短类型名**：用于结果页主标题、分享卡主标题、列表和代码数据。
- **场景长名**：用于结果页副标题、图卡画面方向和 image2 生成提示词。

本文件记录当前暂定的 16 种短类型名。后续如果实现、图卡或文案与本文冲突，以本文为当前命名口径。

## 命名边界

- “长发男”指精神气质，不限定真实发型、性别或身份。
- 产品仍然是娱乐测试，不包装成心理诊断、人格科学或权威测评。
- 结果文案允许锋利、毒舌和轻微冒犯，但不做现实身份羞辱，不引导自伤，不制造诊断结论。
- 短类型名优先考虑移动端可读性、截图传播和亚文化识别度。

## 使用约定

- 结果页 H1、分享卡主标题和结果列表统一使用“短类型名”。
- 结果页副标题、长描述、随机文案池和图卡 prompt 可以继续使用新版 DOCX 中更长的场景表达。
- `AHBG` 已从“河边晒太阳长发男”调整为“河边晒衣长发男”，判词后续可按“晒衣”意象微调。
- `OTDG`、`OTBF`、`OHDF` 已采用更锋利的新短名，后续文案应同步保持这个方向。
- 若后续新增商业化、联名版或付费报告，短类型名保持稳定，扩展文案另行分层。
