## 内容边界

- 本产品是娱乐向、哲学气质映射型网页测试，不是心理诊断、人格科学或权威测评。
- “长发男”指精神气质，不限定真实发型、性别或身份。
- 结果文案可以锋利、损、轻微冒犯，但只冒犯用户的嘴硬、自我保护和拧巴，不冒犯现实身份、性别、阶层、智力、审美或身体状态。
- 所有结果表达默认语义是“你当前更容易这样处理世界”，避免写成“你就是这种人”。
- “危险倾向”是娱乐化提醒，不写成病理、风险评估或处方建议，不出现自伤引导和诊断语气。

## 结果页字段建议

- 短类型名：结果页 H1、分享卡主标题、列表、结果码索引用。
- 场景长名：结果页副标题、图卡画面方向、分享卡副标题用。
- 一句话判词：结果页首屏大字，优先使用 `result-type-naming.md` 当前版本。
- 结果页长描述：首屏后展开，解释气质，不做诊断。
- 随机文案池：结果页尾部、分享图卡、加载过渡和转发按钮附近抽取。
- 危险倾向：保留 2-3 条，做成“你容易把某种优势用过头”的锋利提醒。
- 精神同伙 / 推荐阅读：保留为低层级内容模块，避免暗示作者本人背书或科学归类。

### ATDF | 天台终极长发男

- 为什么叫这个名字：ATDF 是荒诞感、真相优先、解构型、自由个体四个维度的极端交叉。“天台”负责极限视角，“终极”负责一路追问到底，强调他站得高、看得冷，也容易把自己逼到只剩清醒。
- 场景长名：在便利店门口怀疑一切的长发男
- 一句话判词：世界很荒诞，但你偏要问到最后。
- 结果页长描述：你习惯先拆，再决定要不要留下。越是完整的说法，你越想看看接缝在哪。你对意义没有天然敬意，对权威更没有。问题是，你很容易把敏锐活成疲惫，把“不信”误用成“不参与”，最后只剩自己和凌晨的便利店灯光互相证明清醒。
- 随机文案池：
  - 真话不一定救命，但你戒不掉。
  - 你把怀疑磨得比刀还顺手。
  - 不是难相处，是太会听出破绽。
- 危险倾向：把敏锐用成自我消耗；拿撤退维持体面；把孤立误认成自由。
- 精神同伙：卡夫卡 / 尼采 / 加缪
- 推荐阅读：审判 / 善恶的彼岸 / 西西弗神话

### ATDG | 末日人文长发男

- 为什么叫这个名字：ATDG 同样看见荒诞和真相，也习惯拆穿话术，但 G 维度让他舍不得把人丢下。“末日”是他对世界的判断，“人文”是他没有放弃的那部分。
- 场景长名：嘴上退群心里还在护人的长发男
- 一句话判词：看清废墟以后，你还是想把火留下来。
- 结果页长描述：你对大词没信心，对具体的人却始终有反应。你能听懂权力和谎言怎么运转，也知道很多善意最后会被系统磨平。可你仍然舍不得让人彻底变硬，于是常年处在“我知道没用”和“但还是得做点什么”之间来回拉扯。
- 随机文案池：
  - 退群声明写了八遍还是没发。
  - 你嘴比心先到场。
  - 你对人类失望，但对人还有耐心。
- 危险倾向：过度承担；把愤怒转成内耗；总在救火却忘了自己也在冒烟。
- 精神同伙：奥威尔 / 王小波 / 加缪
- 推荐阅读：1984 / 沉默的大多数 / 反抗者

### ATBF | 冷静造船长发男

- 精神同伙：加缪 / 黑塞 / 王小波

### AHDF | 雾中独行长发男

  - 嘴上翻篇，脑子仍在存档。

- 精神同伙：卡夫卡 / 杜拉斯 / 佩索阿
- 推荐阅读：城堡 / 情人 / 不安之书

### AHDG | 温柔犬儒长发男

- 为什么叫这个名字：AHDG 的锋利藏在玩笑里。荒诞感和解构让他容易犬儒，安顿优先和共同体又让他始终舍不得变硬，所以“温柔犬儒”比单纯“丧”更准确。

- 随机文案池：

  - 开玩笑，是你的缓冲层。
  - 看起来很松，实际拧得很紧。
- 危险倾向：用幽默回避求助；替别人兜情绪；把自己的需求讲得像笑话。
- 精神同伙：王小波 / 加缪 / 木心
- 推荐阅读：沉默的大多数 / 局外人 / 云雀叫了一整天

- 为什么叫这个名字：AHBF 不沉迷终极答案，也不等外部救援。他的自救不宏大，常常就是下楼、买点东西、把今晚安顿过去。“便利店”提供具体、廉价、夜间还亮着的生存感。

- 随机文案池：

- 精神同伙：黑塞 / 史铁生 / 川端康成
- 推荐阅读：悉达多 / 我与地坛 / 雪国

### AHBG | 河边晒衣长发男

- 精神同伙：汪曾祺 / 黑塞 / 史铁生

### OHDF | 体面反骨长发男

- 精神同伙：王小波 / 奥威尔 / 加缪

### OHBF | 小院掌灯长发男

- 为什么叫这个名字：OHBF 的关键词是自我边界、生活秩序和小范围建构。“小院”把格局收回可掌控的日子，“掌灯”说明他不是躺平，而是在认真维护自己的亮处。
- 场景长名：在自己小院里拧紧生活螺丝的长发男
- 一句话判词：你不追终极答案，只想把自己的生活点亮。
- 结果页长描述：你喜欢有边界、有节奏、有基本秩序的生活。不是因为你保守，而是因为你知道日子一旦松掉，很容易一路散架。你不迷恋极端体验，也不急着定义自我，更愿意把热水烧开、把账算明白、把该修的东西修掉。安稳在你这儿不是无聊，是功夫。
- 随机文案池：
  - 你把普通日子过得有内部结构。
  - 没有大招，只有稳定续航。
  - 世界乱着呢，你先把抽屉合上。
- 危险倾向：容易压抑冲动；把安全感建得过窄；遇变化时僵住。
- 精神同伙：梭罗 / 汪曾祺 / 史铁生
- 推荐阅读：瓦尔登湖 / 人间草木 / 病隙碎笔

### OHBG | 人间烟火长发男

- 一句话判词：你对世界有基本信心，也愿意照顾具体的人。
- 结果页长描述：你相信秩序该服务于生活，而不是把生活驯成表格。你对关系、日常和审美都很敏感，知道一顿饭、一块旧桌布、一次散步，也可以构成抵抗粗糙现实的方式。你未必爱争论，但很会用生活本身表达立场：不喧哗，不将就，不把自己活成模板。
- 随机文案池：
  - 你拿日常当作品做。
  - 不用高声，生活方式已经表态。
  - 你把烟火气养出了锋芒。

## image2 图卡 prompt 统一规则

### 统一风格

- 画幅固定为 `1080x1350`，优先服务小红书 / 朋友圈竖图传播。
- 主体为成年、东亚语境、长发或中长发的青年形象，可以 male-coded，也可以轻微中性化，但不要强化真实性别身份。
- 气质关键词固定：underground zine、mobile editorial poster、film grain、xerox texture、halftone、indie documentary still、room for bold Chinese title。
- 每张图必须预留中文标题区，标题区不要被脸、手、烟雾、强光或复杂背景压住。
- 允许便利店、楼道、河边、出租屋、桥、书桌、小厨房、公交站、旧办公室等具体场景；少用纯抽象背景。
- 色彩按类型区分，但整体保持低饱和、颗粒感、深夜感，不要做廉价霓虹、赛博算命、二次元美少女或商业海报棚拍感。

### 安全和合规边界

- 不生成真实公众人物、作者肖像、明星脸或可识别品牌 Logo。
- 不出现自伤、自杀边缘、血腥、暴力、药物滥用、未成年人性化、仇恨符号。
- 可以用打火机、烟盒、未点燃香烟、药板、水杯等作为情绪道具，但不要把吸烟、药物或痛苦表现成值得模仿的姿态。
- `天台终极长发男` 的图卡不要站在天台边缘，不要出现坠落暗示；如果用高处，只能是安全的城市远景或屋顶生活场景。
- 图卡不写“心理诊断”“人格类型”“真实人格”“精准测出”等字样。

### 通用 prompt 模板

```text
1080x1350 mobile editorial poster, adult East Asian long-haired youth, male-coded but slightly androgynous, [type-specific scene], [type-specific props], [palette], expression: [type-specific mood], underground Chinese zine aesthetic, film grain, xerox texture, rough halftone, cinematic low light, room for bold Chinese title area, no celebrity likeness, no brand logo, seed {seed}
```

## 仍建议回总控 / 用户讨论的点

1. `ATDF 天台终极长发男`：名称很有记忆点，但“天台”在中文语境里有自伤联想。建议保留短名时，图卡和长描述必须避开天台边缘、跳下、坠落等画面。
2. `AHDF 雾中独行长发男`：名字很稳，但锋利度比其他类型低。如果用户更重视截图传播，可讨论是否要在判词或随机池里补一点刺。
3. `OTDG 漂亮话稽查长发男`：很鲜明，但“稽查”会带一点执法感。当前可保留，后续若用户反馈太硬，再考虑换成更口语的“漂亮话验货”方向。
4. `OTBG 修桥长发男`：短名非常干净，也可能略普通。建议先保留，因为它和“场景长名”形成反差，结果页可以靠长名和文案补锋利。
5. `OHDG 老派温柔长发男`：“老派”应被解释为审美和责任感，不要写成年龄、性别角色或传统道德规训。
6. `OHBG 人间烟火长发男`：传播辨识度略泛，但作为 16 型暖端收束很顺。若后续分享率低，可优先改分享卡副句，不急着改短名。
