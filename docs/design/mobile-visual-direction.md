## 适用范围

本轮视觉参考优先约束：

- 首页首屏
- 答题页 / 问题页

结果页、分享卡和 16 型结果图卡应继承同一视觉语言，但具体构图仍以后续 UI/Frontend 方案为准。

## 视觉关键词

- 黑色旧纸底、低饱和纹理、颗粒感、复印 / 胶片 / 地下刊物质感。
- 大号粗粝中文标题，带旧印刷破损感。
- 红色作为主要强调色，用于主按钮、进度、题号和少量线团装饰。
- 旧唱片、胶带、纸片、线团、涂鸦、档案感边角元素作为页面装饰。
- 布局以移动端竖屏为主，整体像一张可交互的亚文化海报，而不是普通 Web 表单。
- 答题页选项需要有实体卡片感和旧纸描边，不做干净的 SaaS 风格组件。

## 边界

- 不使用廉价赛博、星座算命、二次元、普通问卷表单或商业海报棚拍感。
- 不把“长发男鉴定器”包装成心理诊断、人格科学、精准识别或权威测评。
- 首页和答题页可以锋利、阴暗、亚文化，但不能出现自伤、坠落、血腥或现实伤害暗示。

## 后续责任

- 总控只记录视觉方向和验收信号。
- 架构 / CTO 负责把该方向纳入 UI/Frontend 路线和验收拆分。
- UI/Frontend 负责具体页面还原、响应式尺寸、素材切分、截图验收和可实现性取舍。
